# RadsenBlazorExample

[![Build RadsenBlazorExample on Main](https://github.com/rfalanga/RadsenBlazorExample/actions/workflows/dotnet.yml/badge.svg)](https://github.com/rfalanga/RadsenBlazorExample/actions/workflows/dotnet.yml)

This is a simple app I've written to prove to myself that I can add the NuGet package Radzen.Blazor to a Blazor project and use it to create a simple CRUD app.

Note: I named the GitHub repo RadsenBlazorExample because I made a mistake in spelling. Oh well.

## .NET version used

.NET 8, which as of this writing is the latest LTS version.

## Dependencies

I add Radzen.Blazor to the project.
